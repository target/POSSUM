# Accepted Devices List

This list of devices along with the service object version are proven to work with Possum.  Anyone wishing to add their device to this list should:
1. Contact an [engineer](https://github.com/target/POSSUM?tab=readme-ov-file#testing) to request a hardware addition
2. Send a sample device and service object for testing
3. An engineer will test the provided hardware and determine whether it can be added to the list
